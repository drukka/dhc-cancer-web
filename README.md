## Installation

Steps for basic start:

    $ npm install
    $ npm run start


Edit configuration files:

    config/settings.js - Change API URL


### Features
* React
* React Router
* Redux
* Redux Connect Decorator
* Webpack
* Webpack-hot-middleware
* Babel
* SwaggerClient
* SwaggerClient attached to Redux Actions
* Async-Await support
* BrowserSync
